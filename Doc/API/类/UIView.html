全局函数

UIView ()
通过这个方法创建一个UIView对象

返回值:

    UIView view

例子:

    local view = UIView();

实例方法

setFrame (x, y, w, h)
设置View的位置和大小

参数:

  • x number x坐标
  • y number y坐标
  • w number 宽
  • h number 高

例子:

     local view = UIView();
 view.setFrame(10,10,100,100);
 --或者
 view.frame(10,10,100,100);
 --或者
 view.frame(10,10); --如果只是传递两个值宽高保持原值
frame ()
获取View的位置和大小

返回值:

  1. number x
  2. number y
  3. number w
  4. number h

例子:

    local x,y,w,h = view.frame();

 --或者
